# CAQE
Producer web page:
https://www.react.uni-saarland.de/tools/caqe/

Source code:
https://github.com/ltentrup/caqe
### Dependency
- rust compilator - available at [rustup.rs](http://rustup.rs/)
### How to install
Attached _caqe-master.zip_ contains state of master branch on Jan 6, 2019
- ```unzip caqe-master.zip```
- ```cd caqe-master/```
- ```cargo build --release```
### How to run
- ```cd target/release```
- ```./caqe --help```
